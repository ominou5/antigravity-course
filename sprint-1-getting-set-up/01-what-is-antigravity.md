# Lesson 01: What Is Antigravity?

---

## The Simple Version

Antigravity is an AI-powered coding environment. You describe what you want to build — in plain English — and Antigravity builds it, edits it, tests it, and explains what it's doing along the way.

Think of it as a highly capable collaborator who never gets tired, never judges you for not knowing something, and can write code in any language.

You don't type code. You describe what you want. Antigravity writes the code.

---

## What's an "IDE"?

IDE stands for *Integrated Development Environment* — a fancy name for the app developers use to write and run code. VS Code, Xcode, IntelliJ — these are all IDEs.

Antigravity is an IDE, but with a crucial difference: the primary interface isn't writing code yourself. It's **talking to an AI** that writes the code for you while you remain in control.

---

## How Is It Different From ChatGPT?

You might have used ChatGPT to ask coding questions before. Antigravity is different in some important ways:

| ChatGPT (web) | Antigravity |
|---|---|
| Lives in a browser tab | Lives alongside your project files |
| You copy/paste code in and out | It reads and edits your actual files directly |
| Can't see your project | Works *inside* your project folder |
| Gives you code to paste | Makes the changes for you (with your approval) |
| Stateless between sessions | Can maintain context across a project |

The key difference is **access**. Antigravity doesn't just suggest — it acts. And you decide whether to approve each action.

---

## How Is It Different From Claude Code or Cursor?

There are several AI coding tools out there. Here's the short version:

- **Claude Code** — runs in your terminal, no visual interface
- **Cursor / Copilot** — AI helpers inside traditional code editors; you still write most of the code yourself
- **Antigravity** — a full IDE where the AI is the primary driver; you direct, it builds

Antigravity is designed for a broader audience than just developers. You can be a complete non-coder, a marketer with an idea, a founder who wants to prototype something, or a developer who wants to move 10x faster.

---

## What Can Antigravity Do?

Here's a taste of what you can ask Antigravity to build or do:

**For beginners:**
- "Build me a personal website with my name and bio"
- "Create a simple to-do list app"
- "Explain what this code does in plain English"
- "Fix this error — here's the message I got"

**For builders:**
- "Build a habit tracker with a Supabase database"
- "Add user authentication to my web app"
- "Connect this app to the GitHub API"

**For power users:**
- "Set up a workflow that automatically commits my changes"
- "Install the Firebase skill and connect it to my project"
- "Search the web for the best way to handle rate limiting and implement it"

---

## What Antigravity Can't Do

It's worth being honest:

- It makes mistakes. Always read what it proposes before approving.
- It doesn't know your taste — you have to tell it what you want
- It works best when you give it clear direction
- Very large or complex existing codebases can be harder to work with
- It will occasionally misunderstand you — that's when you clarify and try again

None of these are showstoppers. They're just reasons to stay engaged rather than walk away and come back.

---

## Is It Safe?

Antigravity shows you what it plans to do before doing it. When it wants to edit a file, run a command, or take an action — it tells you. You approve or deny.

You are always in control. Nothing happens without your knowledge.

---

## Summary

- Antigravity is an AI-powered IDE: describe what you want, it builds it
- Works directly with your project files — no copy/pasting
- Designed for builders of all experience levels, from zero to expert
- You stay in control — it asks before making changes
- It makes mistakes; that's normal and fixable
