# Lesson 02: Installation & Your First Launch

---

## What You'll Need

Before you install Antigravity, make sure you have:

- A computer running **Windows, macOS, or Linux**
- An internet connection
- About 10 minutes

That's it.

---

## Step 1: Download Antigravity

1. Go to [antigravity.google](https://antigravity.google) *(or wherever your invite/download link points)*
2. Download the installer for your operating system
3. Run the installer and follow the prompts

> **On Windows:** You may see a security warning saying the app is from an "unknown publisher." This is normal for newer apps. Click **"Run anyway"** or **"More info → Run anyway"** to proceed.

> **On macOS:** If you see a warning about an unverified developer, go to **System Settings → Privacy & Security** and click **"Open Anyway"**.

---

## Step 2: Sign In

When Antigravity opens for the first time, you'll be prompted to sign in or create an account.

- If you already have an account, sign in
- If not, create one — it only takes a minute

Sign in is required because Antigravity uses AI models that run in the cloud. Your account keeps track of your usage.

---

## Step 3: Open a Folder

Antigravity works inside a **folder** (also called a directory or workspace). Think of it as the container for your project.

On your first launch:
1. Click **"Open Folder"** (or similar prompt)
2. Navigate to wherever you want to keep your projects
3. Create a new folder called something like `my-first-project`
4. Open it

You'll now see Antigravity's main interface with your empty folder ready to go.

---

## Understanding the Interface

Antigravity has a few main areas:

```
┌─────────────────────────────────────────────────────┐
│  File Explorer (left)  │  Editor / Preview (center) │
│                        │                             │
│  Your files appear     │  Files you open appear      │
│  here                  │  here for viewing           │
│                        │                             │
├────────────────────────┴─────────────────────────────┤
│  Chat Panel (bottom or side)                         │
│                                                      │
│  > This is where you talk to Antigravity             │
└─────────────────────────────────────────────────────┘
```

The **chat panel** is the most important part. That's where everything happens.

---

## Antigravity Costs — What You Need to Know

Antigravity is not free to run indefinitely, and it's worth understanding how costs work before you dive in.

**How it's billed:**
- Antigravity uses AI models that consume computing resources with each response
- Usage is typically measured in **tokens** (roughly: words in + words out)
- You'll have a **free tier** with a monthly allowance — enough to follow this entire course
- Paid plans unlock higher limits and more powerful models

**What uses tokens:**
- Every message you send and receive uses tokens
- Larger tasks (reading big files, generating lots of code) use more tokens
- Asking the same question twice costs twice as much

**Tips to avoid burning through your quota:**
- Be specific in your prompts — vague prompts lead to long back-and-forths
- Work in focused sessions rather than leaving Antigravity idle
- Use shorter context when possible — don't paste in giant files unless needed
- Check your usage in **Settings → Usage** (or equivalent) to track where you stand

**Where to check pricing:**
Always check [antigravity.dev/pricing](https://antigravity.dev) for the current plan details — pricing can change and it's better to read it directly than rely on a course that might be out of date.

> **Bottom line:** The free tier is generous enough to complete this course. If you're building seriously, a paid plan is worth it for the increased speed and capacity.

## Try It Yourself

Type this in the chat panel and press **Enter**:

```
Hello! I'm new to Antigravity. Can you tell me what you can see in my workspace right now?
```

Antigravity will look at your current folder and describe what's there. If the folder is empty, it'll tell you that too — and probably offer to help you start something.
