# Lesson 06: Building Your First Web Page

---

## What You'll Build

By the end of this lesson, you'll have a real web page — a simple personal page with your name, a short bio, and a list of your interests. It will look good and open in any browser.

You won't write a single line of code. You'll just describe what you want.

---

## How Web Pages Work (The 30-Second Version)

You don't need to memorize this — just have it in the back of your mind:

- **HTML** defines the *content* — headings, text, lists, images
- **CSS** defines the *appearance* — colors, fonts, spacing, layout
- **JavaScript** defines the *behavior* — things that happen when you click or type

When you ask Antigravity to build a web page, it creates these files for you and wires them together.

---

## Your First Page — The Starting Prompt

Open Antigravity with a fresh workspace (or use the one from Lesson 05). Then type:

```
Build me a simple personal page with:
- My name as the main heading
- A short paragraph bio (you can make up placeholder text for now)
- A bulleted list of 3-5 interests
- A modern, clean design with a dark background and white text

Save it as index.html with an accompanying styles.css file.
```

Antigravity will create the files and show you a preview. Approve the changes.

---

## Open It in Your Browser

After Antigravity creates the files:

1. Find `index.html` in your file explorer
2. Right-click → "Open with" → your browser (Chrome, Safari, Firefox, etc.)

You'll see your personal page live in the browser. No server needed — HTML files open directly.

---

## Iterating — Making It Better

This is where it gets fun. Once you see the page, start refining it:

**Change the content:**
```
Update the bio section with this text: [your actual bio]
And replace the placeholder interests with: hiking, building apps, 
coffee, reading sci-fi, and learning languages.
```

**Change the visual design:**
```
The design looks fine but I want it to feel more premium. 
Use a Google Font called "Inter" and increase the spacing between sections.
Add a subtle gradient to the background instead of a flat color.
```

**Add something new:**
```
Add a simple contact section at the bottom with a link to 
my email address: [your@email.com]
Make it look like a button.
```

Each of these is a separate request. You're having a conversation, not writing code.

---

## The Feedback Loop

Building with Antigravity follows a loop:

1. **Ask** for a feature or change
2. **Preview** — review what it proposes
3. **Approve** — apply the change
4. **Check** — open the file in your browser and see how it looks
5. **Refine** — if something's off, describe it and go back to step 1

This loop is the same whether you're building a personal page or a complex app. Get comfortable with it now.

---

## Common Refinement Prompts

Here are prompts you'll use over and over for any project:

```
The spacing between sections feels too tight. Add more breathing room.
```

```
The font is hard to read. Make it larger and use a cleaner typeface.
```

```
The button doesn't stand out enough. Make it bolder with more contrast.
```

```
On my phone the layout breaks — the text is too wide. 
Make the page mobile-friendly.
```

```
I want to add a profile photo. Create a placeholder circle where 
a photo would go, and tell me how to replace it with a real image later.
```

## Try It Yourself

Build your personal page using these prompts in order:

**Step 1 — Basic structure:**
```
Build a personal page with my name, a bio, and a list of interests. 
Dark background, white text, clean modern design. Save as index.html + styles.css.
```

**Step 2 — Add personality:**
```
Add a section called "What I'm Working On" with a brief description 
of a project or goal I'm currently pursuing.
```

**Step 3 — Polish:**
```
Review the overall design of my page and suggest 2-3 improvements 
that would make it look more professional. Then apply them.
```

**Step 4 — Mobile:**
```
Make sure this page looks good on a phone screen. 
Preview the mobile layout for me and fix any issues.
```
